# GamersHub Claude Folder

This folder contains shared Claude-facing guidance for the GamersHub project.

Intentionally included:
- reusable project rules
- task commands
- lightweight skills
- agent role briefs
- design-system guidance based on the current GamersHub Figma direction
- copy-ready prompt templates for Claude in VS Code

Intentionally not included yet:
- `mcp.json`
- `.claude/settings.json`
- executable hooks

Reason:
- those files are valuable only when backed by real, verified IDE/tool configuration
- placeholder configs often mislead agents more than they help
- add them later only after confirming the exact Claude Code / Together IDE format you want to use
